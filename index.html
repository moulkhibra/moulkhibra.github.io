DEFENSE TOOL
SENTINEL v2.0

Real-time cybersecurity defense dashboard for Linux systems. Pulls live system data via psutil, detects brute force attacks, blocks malicious IPs via iptables, and displays everything in a Red Alert terminal UI.

  • Brute force detection via journalctl analysis (every 15s)
  • AbuseIPDB integration for IP reputation checking
  • One-click IP blocking via iptables from the dashboard
  • GeoIP — country/city/ISP for every connection
  • Visual + audio alerts for new threats
  • Attack history saved to JSON with full details
  • Real-time CPU, memory, disk, network monitoring
